然后对比什么
 
对比维度	关注什么
训练损失下降速度	哪个更快收敛？
最终训练损失	哪个更低？
评估回报	哪个最终效果更好？
训练稳定性	哪个的损失曲线更平滑，没有剧烈震荡？
梯度爆炸/消失	哪个更容易出现NaN？
 
判断标准
 
如果归一化后收敛更快、最终效果更好：说明状态各维度量级差异大，归一化有帮助
如果两者差不多：说明当前数据的各维度量级本来就不大，归一化意义不大
如果归一化后反而更差：可能是因为归一化把某些有区分度的小信号压没了
